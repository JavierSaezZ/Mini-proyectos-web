# Mis Proyectos Pequeños

Este repositorio contiene una colección de pequeños proyectos, pruebas de concepto y utilidades.

## Directorio de proyectos
* **[Apuntador](./Apuntador):** Aplicación web para registrar anotaciones y visualizarlas de forma estructurada en una tabla.
* **[Notas Arrastrables](./Notas-Arrastables):** Página interactiva con notas arrastrables (*drag and drop*) y un diseño temático inspirado en gatos.
* **[Ikea Imagenes](./Ikea-Imagenes):** Componente con *hotspots* interactivos sobre imágenes para previsualizar detalles y precio de productos, estilo catálogo de IKEA.
* **[Imagenes video](./Imagnes-video):** Sistema para extraer y mostrar capturas exactas de un vídeo en el instante de reproducción actual.
* **[Interfaz de video](./Interfaz-de-video):** Diseño y maquetación de una interfaz de usuario personalizada para un reproductor de vídeo.
