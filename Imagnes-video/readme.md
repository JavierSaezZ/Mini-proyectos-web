# 📸 Captura de Fotogramas de Vídeo

## 📌 Descripción
Prueba de concepto interactiva que permite extraer y guardar fotogramas exactos de un reproductor de vídeo. Mediante un simple botón, el script captura el instante de reproducción actual, lo procesa utilizando la API de Canvas de HTML5 y genera una imagen en miniatura (escalada al 25%) que se muestra dinámicamente en la pantalla.

## 🚀 Cómo ejecutarlo

Este proyecto funciona íntegramente del lado del cliente en el navegador, por lo que no requiere de un servidor local como WAMP.

Para probar su funcionamiento, sigue estos pasos:

1. Clona o descarga este repositorio en tu ordenador.
2. Asegúrate de colocar un archivo de vídeo cualquiera llamado `video.mp4` en la misma carpeta raíz del proyecto (necesario para que el reproductor tenga contenido).
3. Haz doble clic en el archivo `index.html` para abrirlo directamente en tu navegador web.
4. Reproduce el vídeo y pulsa el botón **"Capture"** en cualquier momento para generar y visualizar las miniaturas del fotograma exacto debajo del reproductor.
